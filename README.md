# ionic-poc-razorpay
ionic-poc-razorpay

### Setup 

```
    cd ./rp_sample
    ionic build
    ionic capacitor add ios
    # note :- you can add android too
    ionic cap run ios -l --external
    
    #android
    ionic capacitor add android
    ionic cap run android -l --external
    
```
### Stack 

- ionic 5 
- capacitor
- angular
